# 🏗️ Барилгын Нягтлан · Construction Accountant

Барилгын нягтлан бодогчид зориулсан **офлайн ажилладаг** туслах апп.
An **offline-first** assistant for construction accountants.

Хэл: **Монгол + English** · Валют: **Төгрөг (₮)** · Интернэт **шаардахгүй**.

---

## 📂 4 үндсэн хэсэг / The four sections

| Хэсэг | Section | Юу хийдэг вэ / What it does |
|---|---|---|
| 📦 **Материал** | Materials | Материалын үнэ, үлдэгдэл, нийлүүлэгчийг бүртгэж **хайх**. Үлдэгдэл багасахад анхааруулна. |
| 📋 **Тооллого** | Stocktaking | Сарын эцэст бодит үлдэгдлийг тоолж, бүртгэлтэй **зөрүүг** өртгөөр гаргана. |
| 💰 **Орлого/Зарлага** | Income & Expense | Мөнгөн орлого, зарлагыг бүртгэж, сар бүрийн дүн, үлдэгдлийг харна. |
| 🏦 **Зээл** | Loans | Авсан/өгсөн зээл, төлбөрийн **хугацааг** хянаж, ойртоход **сануулна**. |

Дээр нь **📊 Хяналтын самбар** (нэгдсэн тойм) ба **⚙️ Тохиргоо** байна.

---

## 🚀 Хэрхэн нээх вэ / How to open

### Арга 1 — Хамгийн хялбар (компьютер дээр)
**`Нягтлан эхлүүлэх.cmd`** файл дээр **давхар дар**. Хөтөч автоматаар нээгдэнэ
(`http://localhost:8080`). Аппыг хаахдаа гарч ирсэн хар цонхыг хаа.

> Double-click **`Нягтлан эхлүүлэх.cmd`** — the browser opens automatically.

### Арга 2 — Шууд нээх
`index.html` файл дээр давхар дарж болно. Интернэтгүй ажиллана, гэхдээ
"утсандаа суулгах" болон мэдэгдэл (notification) ажиллахгүй.

### Арга 3 — Утсандаа суулгах (online)
Утаснаас ашиглах, эсвэл "апп" болгож суулгахын тулд файлуудыг интернэтэд тавина:
1. Компьютер дээрээ `https://app.netlify.com/drop` хаягийг нээ.
2. `construction-accountant` **фолдерийг бүхэлд нь** хуудас руу чирж тавь.
3. Гарч ирсэн холбоосыг утасныхаа **Chrome**-оор нээ → цэс (⋮) → **«Апп суулгах»**.

## 📝 Хэрэглэх заавар / How to use

1. **Эхлээд материалаа бүртгэ** (📦 Материал → ＋ Материал нэмэх): нэр, нэгж үнэ,
   эхний үлдэгдэл, доод хязгаар. Доод хязгаараас бага бол улаанаар анхааруулна.
2. **Худалдан авалт хийхэд** (💰 Орлого/Зарлага → ＋ Гүйлгээ нэмэх → Зарлага):
   материалаа сонгоод тоо хэмжээгээ оруулбал **үлдэгдэл автоматаар нэмэгдэнэ**.
3. **Сарын эцэст** (📋 Тооллого → ＋ Шинэ тооллого): бодит тоог оруулаад
   "Тооллого хадгалах". Зөрүүг харсны дараа "Үлдэгдэлд буулгах" дарж бодит
   тоог системд тааруулна.
4. **Зээл** (🏦 → ＋ Зээл нэмэх): үндсэн дүн, төлөх огноог оруул. Төлбөр хийх бүрд
   "Төлбөр нэмэх". Хугацаа ойртоход Хяналтын самбар дээр сануулна.

## 💾 Чухал: Өгөгдлийн нөөц / Important: back up your data

Бүх өгөгдөл **зөвхөн энэ төхөөрөмжид** хадгалагдана (сервер байхгүй).
Алдахгүйн тулд тогтмол нөөшил:

**Тохиргоо → Нөөшлөх → «Өгөгдөл татах (JSON)»** — файл татагдана.
Сэргээх: **«Өгөгдөл оруулах»** → тэр JSON файлаа сонгоно.

> Шинэ утас/компьютер руу шилжихдээ энэ JSON-оор бүх өгөгдлөө зөөнө.

## 🔧 Техникийн мэдээлэл / Technical notes

- Цэвэр HTML/CSS/JavaScript — суулгац (build) шаардахгүй.
- Өгөгдөл хадгалалт: хөтчийн `localStorage` (түлхүүр: `ca_v1`).
- PWA: `manifest.webmanifest` + `sw.js` (офлайн кэш, мэдэгдэл).
- Хуучин дуу-хоолойт апп **`_archive_voice_app`** фолдерт хадгалагдсан.
  Шинэ апп нь хуучин өгөгдлийг анх ачаалахдаа автоматаар импортлохыг санал болгоно.
