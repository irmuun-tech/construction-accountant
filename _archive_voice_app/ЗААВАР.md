# Барилгын Нягтлан — Суулгах заавар

Энэ бол таны утсан дээр ажиллах **дуу хоолойн туслах** апп юм:
монголоор ярихад текст болж, материал/орлого бүртгэж, сар бүр Excel гаргадаг.

---

## 🚀 Утсандаа суулгах (3 алхам, ~3 минут)

Апп интернетэд байрлах ёстой (микрофон, мэдэгдэлд HTTPS шаардлагатай).
Хамгийн хялбар арга нь **Netlify Drop** — бүртгэл, төлбөргүй:

### 1. Файлуудаа байршуулах
1. Компьютер дээрээ `https://app.netlify.com/drop` хаягийг нээ.
2. `C:\Users\Dell\construction-accountant` **фолдерийг бүхэлд нь** хуудас руу **чирж тавь** (drag & drop).
3. Хэдхэн секундын дараа `https://нэр-нэр.netlify.app` гэсэн **холбоос** гарч ирнэ. Үүнийг хуулж ав.

### 2. Утсан дээрээ нээх
1. Утасныхаа **Chrome** хөтчөөр тэр холбоосыг нээ.
2. Цэс (︙) → **«Нүүр дэлгэцэнд нэмэх» (Add to Home screen)** дар.
3. Одоо нүүр дэлгэцэнд жинхэнэ апп шиг дүрс гарч ирнэ. 🏗️

### 3. Зөвшөөрөл өгөх
- Эхний удаа 🎤 товч дарахад **микрофон** зөвшөөрөл асууна → «Зөвшөөрөх».
- Эхний сануулга нэмэхэд **мэдэгдэл** зөвшөөрөл асууна → «Зөвшөөрөх».

> Дараа нь апп шинэчлэх бол: Netlify Drop руу фолдероо дахин чирж тавихад л болно.

## 📱 Хэрхэн ашиглах

| Таб | Юу хийдэг вэ |
|-----|-------------|
| **➕ Бичих** | 🎤 дараад ярь. Текст, дүн автоматаар орно. **Шалгаад** «Хадгалах». Материал/Орлого хооронд солих. |
| **📋 Жагсаалт** | Тухайн сарын бүх бичлэг, нийт орлого, материалын зардал. ✕-р устгана. |

## ❓ Асуудал гарвал

- **Дуу таних ажиллахгүй бол:** заавал **Chrome** ашигла (Samsung Internet зэрэг
  бусад хөтөч дэмжихгүй). Дуу таних үед интернет хэрэгтэй.
- **Нөөц арга:** аль ч бичих талбарт хүрээд, гар (Gboard) дээрх **микрофон** товчоор
  монголоор бичиж болно — энэ нь үргэлж ажилладаг.
- **Өгөгдөл хаана хадгалагддаг вэ?** Таны утсанд (бусдад илгээгддэггүй). Тийм болохоор
  Excel-ээ сар бүр татаж, нөөцлөж байхыг зөвлөе.

## ☁️ Үүлэн бүртгэл тохируулах (нэвтрэлт + нөөцлөлт)

Энэ нь өгөгдлийг үүлэнд хадгалж, олон утаснаас нэвтрэх боломж олгоно.
Firebase нь Google-ийн **үнэгүй** үйлчилгээ. Нэг удаа тохируулна:

1. `https://console.firebase.google.com` руу ороод Google хаягаараа нэвтэр.
2. **«Add project»** → нэр өг (ж: `nyagtlan`) → үргэлжлүүл (Analytics хэрэггүй).
3. Зүүн дээд талын **</> (Web)** дүрс дээр дарж апп нэмэх → нэр өг → **Register**.
4. Гарч ирэх `firebaseConfig` блокоос **6 утгыг** хуулж ав (apiKey, authDomain, ...).
5. **Authentication** → **Get started** → **Email/Password** → **Enable** → Save.
6. **Firestore Database** → **Create database** → **Start in production mode** → бүс сонгоод үүсгэ.
   Дараа нь **Rules** таб руу ороод доорхийг тавиад **Publish** дар:

   ```
   rules_version = '2';
   service cloud.firestore {
     match /databases/{database}/documents {
       match /users/{uid}/{document=**} {
         allow read, write: if request.auth != null && request.auth.uid == uid;
       }
     }
   }
   ```

7. Компьютер дээрх `firebase-config.js` файлыг нээж, 6 утгыг өөрийнхөөрөө сольж хадгал.
8. Аппаа дахин байршуулаад (Netlify Drop), 👤 товчоор имэйл/нууц үгээр **бүртгүүл**.

> Тохируулаагүй байсан ч апп офлайн горимд хэвийн ажиллана.

«Бичих» хэсэгт материалын нэр бичихэд (ж: *цемент*, *тоосго*, *арматур*)
интернэтээс зураг автоматаар хайж харуулна. Тохирох зургийг дарж сонгоход
тухайн бүртгэлд хадгалагдана. (Зургийн хайлтад интернэт хэрэгтэй.)
